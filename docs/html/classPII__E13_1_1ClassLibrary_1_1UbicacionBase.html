Detailed Description

Representa a la implementación básica del tipo IUbicacion, incluyendo implementaciones para todas sus operaciones. Aplica Expert ya que se le da la responsabilidad de gestionar las coordenas de una ubicacion, debido a que es la clase más experta de la información.
