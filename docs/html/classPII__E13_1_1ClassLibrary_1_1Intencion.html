Detailed Description

Representa una intención obtenida a través de procesamiento de lenguaje natural (PLN) a partir de un texto. Es una clase contenedora de información. Se aplica el patrón Adapter para desacoplar las APIs de Google para PLN a través de esta clase y su clase constructora. LenguajeNatural, que representa a un procesador de lenguaje natural.
